# Plan: Dashboard Utama Rental Mobil

## Overview
Membuat halaman dashboard utama di `/dashboard` yang berisi sidebar navigasi, ringkasan statistik, dan bagian transaksi aktif.

## Project Context
- **Framework**: Next.js 16.2.7 + React 19 + TypeScript
- **Styling**: Tailwind CSS 4 (v4 ŌĆö `@theme inline`)
- **Path alias**: `@/*` -> `./*`
- **Status**: Fresh project, belum ada komponen/shadcn/ui
- **Icon library**: Akan pakai `lucide-react` (shadcn default)

## Implementation Steps

### 1. Init shadcn/ui + Install Components
```bash
npx shadcn@latest init -y
npx shadcn@latest add sidebar card badge table
```

### 2. Buat Sidebar Component (`components/app-sidebar.tsx`)
Sidebar dengan menu navigasi:
- **Dashboard** (ikon `LayoutDashboard`) ŌĆö `/dashboard`
- **Daftar Armada** (ikon `Car`) ŌĆö `/dashboard/armada`
- **Transaksi** (ikon `Receipt`) ŌĆö `/dashboard/transaksi`
- **Pelanggan** (ikon `Users`) ŌĆö `/dashboard/pelanggan`

Gunakan shadcn Sidebar component. Tampilkan logo "Rental Mobil" di header sidebar.

### 3. Buat Dashboard Layout (`app/dashboard/layout.tsx`)
Layout yang membungkus semua route `/dashboard/*`:
```tsx
<SidebarProvider>
  <AppSidebar />
  <main>
    <SidebarTrigger />
    {children}
  </main>
</SidebarProvider>
```

### 4. Buat Dashboard Page (`app/dashboard/page.tsx`)
Halaman utama berisi:

**a) Ringkasan Statistik (4 Card)**
- Total Armada: 24 unit
- Mobil Disewa: 18 unit
- Pendapatan Bulan Ini: Rp 45.200.000
- Total Pelanggan: 156

**b) Transaksi Aktif (Table)**
Menampilkan tabel transaksi dengan kolom:
- No
- Pelanggan
- Mobil
- Tanggal Sewa
- Status (Badge: Disewa/Dikembalikan/Jatuh Tempo)

Data dummy statis (hardcoded untuk demo).

### 5. Update Root Layout
- Update title metadata ke "Rental Mobil"
- Redirect `/` ke `/dashboard` (via Next.js redirect atau simple re-export)

### 6. Update globals.css
Tambahkan CSS variables untuk shadcn/theme jika belum ada.
