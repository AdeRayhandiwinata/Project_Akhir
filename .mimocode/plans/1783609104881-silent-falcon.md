# Rencana: Membuat PPT Presentasi dari Project Rental Mobil

## Ringkasan
Membuat file PowerPoint (.pptx) menggunakan library `pptxgenjs` yang menghasilkan presentasi akademik lengkap untuk project "DriveRent" / "Rental Mobil" berbasis Next.js.

## Teknologi
- **Library:** `pptxgenjs` — generate .pptx secara programmatic
- **Bahasa:** JavaScript/TypeScript (Node.js script)
- **Output:** `rental-mobil-presentation.pptx`

## Struktur Slide (8 Slide, Bahasa Indonesia)

### Slide 1: Cover / Judul
- Judul: "Sistem Informasi Rental Mobil Berbasis Web"
- Subjudul: "DriveRent — Website & Dashboard Admin"
- Nama, NIM, Mata Kuliah, Tahun Ajaran
- Template: gelap/profesional

### Slide 2: Latar Belakang
- Permasalahan: manajemen rental mobil masih manual
- Solusi: sistem berbasis web yang memudahkan pencarian, pemesanan, dan pengelolaan armada
- Target pengguna: pelanggan umum (website) & admin (dashboard)

### Slide 3: Tech Stack
- Framework: Next.js 16 (App Router)
- Language: TypeScript
- UI: React 19, Tailwind CSS v4, shadcn/ui
- Icons: Lucide React
- Font: Geist Sans + Geist Mono
- Diagram/simetri stack

### Slide 4: Fitur Website (Public)
- Navbar (responsive, mobile menu)
- Hero Section (CTA + WhatsApp)
- CarList (grid 6 mobil dengan detail harga, kursi, bahan bakar)
- HowItWorks (3 langkah sewa)
- Testimoni pelanggan
- Footer (jam operasional, kontak)
- Integrasi WhatsApp untuk booking

### Slide 5: Fitur Dashboard Admin
- Sidebar navigation ("Balikpapan Trans")
- Dashboard Overview: Total Armada (24), Transaksi Hari Ini (8), Pelanggan Aktif (156), Pendapatan Bulan (Rp 45JT)
- Modul Armada: CRUD + filter + search
- Modul Pelanggan: CRUD + statistik
- Modul Transaksi: CRUD + status tracking

### Slide 6: Arsitektur / Struktur Code
- Tree struktur folder (app/, components/, hooks/, lib/)
- Alur data: Hardcoded mock data → React components → Tailwind styling
- App Router: layout.tsx → page.tsx per modul
- Diagram alur komponen

### Slide 7: Screenshot / Demo
- Screenshot dari running app: landing page, dashboard, halaman armada, transaksi
- (Catatan: perlu jalankan app dulu untuk ambil screenshot, atau gunakan deskripsi teks)

### Slide 8: Kesimpulan & Saran
- Kesimpulan: project berhasil membangun sistem rental mobil berbasis web dengan fitur lengkap
- Saran: tambahkan backend/database, autentikasi user, payment gateway, mobile app

## File yang Dibuat
- `generate-ppt.js` — script Node.js untuk generate PPT
- `rental-mobil-presentation.pptx` — output PPT

## Langkah Eksekusi
1. Install `pptxgenjs` (`npm install pptxgenjs --save-dev`)
2. Buat script `generate-ppt.js` dengan semua konten slide
3. Jalankan script untuk generate file .pptx
4. Verifikasi file terbuka dengan benar

## Verifikasi
- Buka file `rental-mobil-presentation.pptx` di Microsoft PowerPoint / Google Slides
- Pastikan semua 8 slide ter-render dengan benar
- Cek teks, format, dan layout
